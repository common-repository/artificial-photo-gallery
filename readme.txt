== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plugin-directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use shortcode in page or post.
4. If you want photo gallery in your theme php, Place `<?php echo do_shortcode('[artificial-gallery]'); ?>` in your templates


<strong><i>Shortcode</i></strong>
<em>Just place this code your page or post.</em>

<pre>[artificial-gallery]</pre>
